S4!NT is a slackbot built using the mission guide in the stardance program for hack club
this slack app runs 24/7 in Nest following the instructions given by the guide https://stardance.hackclub.com/missions/slack-bot/guide
the bit includes the following commands:
/s4nt-ping
Checks if S4!NT is online
/s4nt-catfact
provides facts about cats
/s4nt-joke
tell the user jokes

This bot will likely be updated in the future to include more features.
